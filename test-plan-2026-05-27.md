# Raven v1.3.0 Test Plan

## TTS 集成 + 新功能测试

### 前置条件

1. 已配置 LLM 模型（设置页）
2. 已配置 TTS（设置页 → TTS 语音设置 → 填写 API URL、Key → 点"测试语音"确认能听到声音）

---

## 一、TTS 基础功能

### 1.1 设置页 TTS 配置

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 进入设置页，找到"TTS 语音设置"卡片 | 显示 API URL、Key、音色、语速输入框 |
| 2 | 填写 API URL 和 API Key | 输入正常 |
| 3 | 点击"测试语音" | 按钮变为加载状态，随后听到 "Hello, this is a test." |
| 4 | 修改语速为 0.5，再次测试 | 语速明显变慢 |
| 5 | 修改语速为 100（超出范围），点击保存 | 自动 clamp 到 4.0 |
| 6 | 点击"保存设置" | 无报错（静默保存） |

### 1.2 词汇发音（VocabularyPage）

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 进入生词本，找到任意单词卡片 | 单词右侧有喇叭图标 |
| 2 | 点击喇叭图标 | 图标变为加载 spinner，随后听到单词发音，播放中图标变为 VolumeX |
| 3 | 播放中再次点击 | 立即停止播放 |
| 4 | 快速连续点击不同单词的喇叭 | 前一个停止，播放新的 |

### 1.3 复习发音（ReviewPage）

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 进入复习，开始复习 | 翻卡正面单词旁有喇叭图标 |
| 2 | 点击正面喇叭 | 播放单词发音，卡片不翻转（stopPropagation 生效） |
| 3 | 点击翻转查看背面 | 背面单词旁也有喇叭图标 |
| 4 | 点击背面喇叭 | 播放单词发音 |

### 1.4 写作对比听（CorrectPage）

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 输入一段有错误的英文，提交批改 | 等待结果返回 |
| 2 | 找到 "Corrected" 标题旁的喇叭图标 | 点击后播放修正后的完整文本 |
| 3 | 找到某条纠错项 | original 和 corrected 文本旁各有一个喇叭 |
| 4 | 点击 original 的喇叭 | 播放错误原文 |
| 5 | 点击 corrected 的喇叭 | 播放修正后的文本，可对比听出差异 |

### 1.5 阅读朗读（ReadingPage）

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 粘贴一段英文文章，开始分析 | 等待六维分析完成 |
| 2 | 找到 "Original" 区域右侧的"朗读"按钮 | 按钮可见 |
| 3 | 点击"朗读" | 开始逐句播放，当前句子在原文中高亮（黄色背景） |
| 4 | 等待自动播放下一句 | 高亮跟随切换到下一句 |
| 5 | 点击"停止朗读" | 立即停止播放，高亮消失，按钮恢复为"朗读" |
| 6 | 朗读过程中切换到其他页面再切回来 | 朗读仍在进行（PersistentRoutes 保持挂载） |
| 7 | 朗读过程中点击"Start Reading"提交新分析 | 旧朗读停止，开始新的分析 |

## 二、听力练习（新功能）

### 2.1 基础流程

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 侧边栏点击"听力练习" | 进入听力练习页面，显示主题输入框和难度选择 |
| 2 | 主题输入"科技"，点击"中级" | 开始生成句子，30 秒后如未完成显示"重新生成"提示 |
| 3 | 生成完成 | 自动播放第一句 TTS |
| 4 | 点击大圆播放按钮 | 重复播放当前句子 |
| 5 | 点击"查看中文提示" | 显示中文提示文字 |
| 6 | 在输入框中输入听到的句子 | 输入正常 |
| 7 | 点击"下一句" | 跳转到下一句并自动播放 |
| 8 | 完成所有句子后点击"提交" | 显示评分结果 |

### 2.2 评分结果

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 查看得分卡片 | 显示 X/5 正确，颜色随分数变化（绿/黄/红） |
| 2 | 查看每句详情 | 正确的句子显示绿色边框，错误的显示红色边框 |
| 3 | 错误句子 | 显示原文（绿色）和你的回答（红色） |
| 4 | 点击任意句子的播放按钮 | 可重新播放该句 |
| 5 | 点击"再来一轮" | 重新生成新句子 |

### 2.3 历史记录

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 完成一轮听力练习后进入历史记录 | 列表中出现一条"Listening"类型记录 |
| 2 | 点击查看详情 | 显示青色主题的得分卡片和逐句对比 |

## 三、语速训练器（新功能）

### 3.1 输入阶段

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 侧边栏点击"语速训练" | 进入语速训练页面，显示文本输入框 |
| 2 | 粘贴一段英文文章 | 底部显示"约 X 个句子" |
| 3 | 点击"开始训练" | 进入播放器界面 |

### 3.2 播放器

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 查看语速按钮组 | 显示 0.5x / 0.75x / 1.0x / 1.25x / 1.5x，1.0x 高亮 |
| 2 | 点击播放按钮 | 开始从第一句播放，当前句高亮 |
| 3 | 播放中点击暂停 | 暂停播放 |
| 4 | 点击下一句按钮 | 跳到下一句并播放 |
| 5 | 点击上一句按钮 | 跳到上一句并播放 |
| 6 | 点击停止按钮 | 停止播放，高亮消失 |
| 7 | 点击句子列表中的某一句 | 直接播放该句 |

### 3.3 语速切换

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 以 1.0x 播放一句 | 正常语速 |
| 2 | 切换到 0.5x | 当前句重新以慢速播放 |
| 3 | 切换到 1.5x | 当前句重新以快速播放 |
| 4 | 切换语速时观察播放按钮 | 短暂显示加载 spinner（TTS 缓存未命中时） |

### 3.4 循环模式

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 点击循环按钮（默认不循环） | 图标变为单句循环（Repeat1） |
| 2 | 再次点击 | 图标变为全文循环（Repeat） |
| 3 | 再次点击 | 恢复不循环 |
| 4 | 单句循环模式下播放 | 当前句播完后重复播放 |
| 5 | 全文循环模式下播放到最后一句 | 自动回到第一句继续播放 |

## 四、回归测试

### 4.1 现有功能不受影响

| 功能 | 验证点 |
|------|--------|
| Writing Copilot | 输入文本 → 批改 → 结果正常显示 |
| Reading Copilot | 粘贴文章 → 六维分析 → 知识图谱正常 |
| 生词本 | 添加/删除/筛选单词正常 |
| 复习 | 翻卡、评分、间隔重复算法正常 |
| 弱项训练 | 生成练习、答题、评分正常 |
| 历史记录 | 各类型记录列表和详情正常 |
| 分析面板 | 图表和弱项推荐正常 |
| 任务状态栏 | 写作/阅读/练习的 running/completed 状态正常 |

### 4.2 边界情况

| 场景 | 验证点 |
|------|--------|
| TTS 未配置 | 点击喇叭按钮无反应（不报错） |
| LLM 未配置 | 听力练习显示"请先配置 LLM 模型" |
| 网络断开 | TTS 请求失败时优雅降级 |
| 快速连续点击 | 不产生音频重叠或状态异常 |
| 页面切换 | 进行中的朗读/播放正确停止或继续 |
